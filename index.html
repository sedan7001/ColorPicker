시작하기

Color Picker를 어떻게 만들었는지, 어떻게 사용하지는지 알아봅니다.

Color Picker는 사용하기 쉽습니다. 색상값이 필요한 모든이를 대상으로 합니다.

Overview

색상값이 필요할 때 정확한 색상을 제공하는 색상 선택기를 만드는 프로젝트 입니다.

Bootstrap 3.3.2
색상 선택기는 신뢰할 수 있는 도구인 부트스트랩을 사용하였습니다.
Responsive
CSS 미디어 쿼리를 이용하여 휴대폰, 태블릿, 데스크탑까지 쉽게 사용이 가능합니다.
Configurable Color Scheme
언제든지 색 구성표의 색을 변경하거나 추가할 수 있도록 개발하였습니다.

Easy to use

누구나 쉽게 사용할 수 있습니다. 이제 원하는 색상값을 복사하기만 하면 됩니다.
